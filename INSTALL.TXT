Installation of PowerAdminUrt
--------------------------------------------------------------------------------
*  Read all in this zip file included readme's prior to installation. 

*  From the zip file :
     - copy the directory extplugins/poweradminurt/ into your b3/extplugins/ directory
     - copy the file extplugins/conf/plugin_poweradminurt.ini into the folder that contains your main b3.xml config file
     - add to the plugins section of your main b3 config file:

     <plugin name="poweradminurt" config="@conf/plugin_poweradminurt.ini" />
